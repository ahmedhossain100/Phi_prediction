Phi Estimation of Silty Sand using SPT-N60 and Grain Size Analysis

    In this study, a Machine Learning model is produced to estimate the angle of internal friction (Phi) of Sility Sand (SM) from SPT-N 60 and the particle size of the soil sample.
    176 sample of Silty Sand from Joydevpur- Mymensingh- Jamalpur section in Bangladesh were used.
    MLR, SVM and ANN were considered during the model preparation where SVM which is used in this web app have shown better performance.
    The toal features considered in the model are STN-N60, Depth of Sample Collection, Sand(%), Silt(%), Clay(%), D10, D30, D60, Cu, and Cc, and the optimum features are selected through feature selection.
    The link of the Github repository is provided.
